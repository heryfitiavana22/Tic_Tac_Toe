# Tic Tac Toe
Link : https://tic-tac-toe-4ufj.onrender.com/

## Features
**we can** :
- play locally
- play online and choose our opponent displayed on *available opponent*


## Project setup 
Clone the project
```
git clone https://github.com/heryfitiavana22/Tic_Tac_Toe.git
```

Install dependency
```
npm install
```
 or 
```
pnpm install
```

Run the server
```
npm run server
```

**Enjoy game  ;)**
